To start and run a Python Django project on Ubuntu, follow these main steps:

1. Install Python, pip, and a virtual environment tool:
   - Update your package list and upgrade existing packages.
   - Install Python 3, pip (Python package manager), and virtual environment support.
2. Create and activate a virtual environment:

   - Create a dedicated directory for your Django project.
   - Set up a Python virtual environment inside this directory.
   - Activate the virtual environment to isolate project dependencies.

3. Install Django within the virtual environment using pip.

4. Create a new Django project using the `django-admin startproject` command.

5. Run database migrations with `python manage.py migrate`.

6. Start the Django development server using `python manage.py runserver`.

Details:

- Run these commands in terminal:
  ```
  sudo apt update && sudo apt upgrade -y
  sudo apt install python3 python3-pip python3-venv -y
  mkdir mydjangoapp && cd mydjangoapp
  python3 -m venv env
  source env/bin/activate
  pip install django
  django-admin startproject myproject .
  python manage.py migrate
  python manage.py runserver
  ```
- Your Django app will be accessible by default at http://127.0.0.1:8000.

- You can configure the project further by editing `myproject/settings.py` (e.g., add allowed hosts).

These steps set up a Django project for local development on Ubuntu using best practices like virtual environments and
pip-installed dependencies.[1][3][5][7]
