Python provides a broad array of built-in string methods that can be used to manipulate and query string data efficiently.
Here are some of the most common Python string methods:

- **capitalize()**: Converts the first character of the string to uppercase.
- **casefold()**: Converts the string to lowercase, useful for caseless comparisons.
- **center(width)**: Returns a string centered in a field of given width.
- **count(substring)**: Returns the number of times a specified substring appears in the string.
- **encode()**: Returns an encoded version of the string.
- **endswith(suffix)**: Checks if the string ends with the specified suffix.
- **find(substring)**: Returns the index of the first occurrence of the substring; -1 if not found.
- **format()**: Allows string formatting by substituting placeholders.
- **isalnum()**: Checks if all characters are alphanumeric.
- **isalpha()**: Checks if all characters are alphabetic.
- **isdigit()**: Checks if all characters are digits.
- **islower() / isupper()**: Checks if all characters are lowercase or uppercase respectively.
- **join(iterable)**: Concatenates elements of an iterable with the string as a separator.
- **lower() / upper()**: Converts the entire string to lowercase or uppercase.
- **replace(old, new)**: Replaces occurrences of a substring with another substring.
- **split(separator)**: Splits the string into a list using the specified separator.
- **strip()**: Removes leading and trailing whitespace from the string.
- **startswith(prefix)**: Checks if the string starts with the specified prefix.
- **title()**: Converts the first character of each word to uppercase.
- **swapcase()**: Swaps uppercase characters to lowercase and vice versa.
- **zfill(width)**: Pads the string with zeros on the left to reach the specified width.

These methods do not alter the original string; instead, they return a new string with the applied changes. They are
fundamental for efficient and readable string handling in Python programming.[1][3][4]

Combined with common built-in Python functions like len(), print(), and str(), these string methods provide powerful tools
for text processing and manipulation.[11][12]
