Here is a detailed description and comparison of Python's `filter`, `map`, `zip`, and `reduce` functions with their
JavaScript/TypeScript equivalents:

## Python Functions

- **filter(function, iterable)**
  - Takes a function and an iterable (like a list).
  - Returns an iterator with elements for which the function returns True.
  - Used for selective filtering of data.
- **map(function, iterable, ...)**
  - Applies the given function to each item in the iterable(s).
  - Returns an iterator with the results.
  - Can take multiple iterables, applying the function on them element-wise.
- **zip(\*iterables)**
  - Takes multiple iterables and returns an iterator of tuples.
  - Each tuple contains elements from the input iterables at corresponding positions.
  - Stops at the shortest input iterable.
- **reduce(function, iterable[, initializer])**
  - From `functools` module.
  - Applies the function cumulatively to the items of the iterable, reducing it to a single value.
  - For example, summing or multiplying all elements.

## JavaScript/TypeScript Equivalents

| Python   | JavaScript/TypeScript            | Notes                                                             |
| -------- | -------------------------------- | ----------------------------------------------------------------- |
| `filter` | `Array.prototype.filter()`       | Similar behavior; filters array elements.                         |
| `map`    | `Array.prototype.map()`          | Applies function to array elements.                               |
| `zip`    | Custom function or use libraries | No native zip function; needs implementation or lodash `_.zip()`. |
| `reduce` | `Array.prototype.reduce()`       | Same concept; reduces array to a single value.                    |

### Summary

- Python's `filter` and `map` correspond directly to JavaScript's `filter()` and `map()` methods on arrays.
- Python's `reduce` is equivalent to JavaScript's `reduce()` array method.
- Python's `zip` has no built-in equivalent but can be implemented via functions combining multiple arrays into tuples or
  arrays.

### Example (Python vs JS)

Python:

```python
nums = [1, 2, 3, 4]
filtered = list(filter(lambda x: x % 2 == 0, nums))  # [2, 4]
mapped = list(map(lambda x: x * 2, nums))            # [2, 4, 6, 8]
zipped = list(zip([1, 2], ['a', 'b']))               # [(1, 'a'), (2, 'b')]
from functools import reduce
sum_all = reduce(lambda a, b: a + b, nums)           # 10
```

JavaScript:

```js
const nums = [1, 2, 3, 4];
const filtered = nums.filter((x) => x % 2 === 0); // [2, 4]
const mapped = nums.map((x) => x * 2); // [2, 4, 6, 8]
const zipped = nums.map((v, i) => [v, ["a", "b"][i]]); // [[1,'a'], [2,'b'], [3, undefined], [4, undefined]]
const sum_all = nums.reduce((a, b) => a + b, 0); // 10
```

Python's `zip` usually requires a custom solution or library in JS to handle unequal length arrays elegantly.
