Here is a list of the most common MIME types used on the web:

- text/plain: Plain text files (e.g., .txt)
- text/html: HTML documents (e.g., .html, .htm)
- text/css: Cascading Style Sheets (CSS) (e.g., .css)
- text/csv: Comma-separated values (CSV) files (e.g., .csv)
- application/json: JSON format files (e.g., .json)
- application/javascript: JavaScript files (e.g., .js)
- image/jpeg: JPEG image files (e.g., .jpg, .jpeg)
- image/png: PNG image files (e.g., .png)
- image/gif: GIF image files (e.g., .gif)
- image/svg+xml: Scalable Vector Graphics files (e.g., .svg)
- application/pdf: PDF documents (e.g., .pdf)
- application/zip: ZIP archive files (e.g., .zip)
- application/octet-stream: Binary data, generic (used for unknown file types)
- audio/midi: MIDI audio files (e.g., .midi)
- audio/mpeg: MPEG audio files (e.g., .mp3)
- video/mp4: MP4 video files (e.g., .mp4)
- video/mpeg: MPEG video files (e.g., .mpeg)
- application/msword: Microsoft Word files (e.g., .doc)
- application/vnd.ms-excel: Microsoft Excel files (e.g., .xls)
- application/vnd.ms-powerpoint: Microsoft PowerPoint files (e.g., .ppt)

This collection covers text, image, audio, video, application document, and archive types commonly encountered in web and
software environments.[2][4][9]
